# 44-4. Compact and Lightweight VR Glasses using Double Path Pancake Optics

1. **제목**
   - Compact and Lightweight VR Glasses using Double Path Pancake Optics

2. **저자**
   - Naru Usukura, Kiyoshi Minoura, Yoshiko Honma

3. **소속**
   - Sharp Display Technology Corporation, Nara, Japan
   - Kantatsu company, Tokyo, Japan

4. **시사점**
   - 샤프(Sharp)는 VR 기기의 소형화와 경량화를 위해 기존 팬케이크(Pancake) 광학계의 낮은 광 효율(약 25%)을 개선한 'Double Path(DP) 팬케이크 광학계'를 제안함.
   - 새로운 광학 설계를 통해 기존 대비 약 1.8배 높은 광 추출 효율을 달성했으며, 이를 통해 디스플레이의 휘도 요구치를 낮추고 소비 전력 및 발열 문제를 동시에 해결함.
   - 텔레센트릭(Telecentric) 광학 시스템을 적용하여 0D에서 -9D까지 넓은 범위의 시력 보정(Diopter adjustment)이 가능하며, 보정 시에도 배율 변화와 화질 저하가 거의 없음.
   - 이 기술을 적용한 'Xrostella VR1' 안경형 VR 기기는 198g의 초경량 무게를 실현하여 착용감을 극대화함.

5. **본기술의 의의 당사 기술수준과 비교**
   - **의의**: VR 광학계의 고질적 난제인 '광 손실'을 혁신적인 경로 설계로 돌파하여, 결과적으로 디스플레이 패널의 전력 효율을 간접적으로 높인 점이 핵심임. 초경량 VR 기기 구현을 위한 광학계와 패널 간의 통합 최적화 사례로서 큰 의미가 있음.
   - **기술수준 비교**: 당사의 XR 기기 전략에서도 팬케이크 광학계는 필수 요소이나, 샤프가 보여준 1.8배의 광 효율 향상은 패널의 휘도 마진을 획기적으로 넓혀줄 수 있는 위협적인 기술임. 당사는 고휘도 OLEDoS 패널 자체의 성능에 집중하고 있으나, 샤프와 같이 광학계 효율 극대화를 통해 전체 세트의 전력 소모를 줄이는 접근 방식은 당사의 패널 설계 시 반드시 연계 검토해야 할 벤치마킹 포인트임. 특히 넓은 범위의 시력 보정 기능은 사용자 편의성 측면에서 당사 차세대 기기 설계에 반영이 시급함.
