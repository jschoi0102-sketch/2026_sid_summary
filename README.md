# SID 2026 Paper Summaries

This repository contains technical summaries, A4-sized PPT slides, and an analysis of research papers presented at the SID 2026 conference.

## Project Objective
The goal is to extract key technical insights from SID 2026 Oral sessions, perform comparative benchmarking against Samsung Display's technology standards, and provide actionable information for strategic planning and presentation preparation.

## Contents
- **Session Summaries**: Individual Markdown files (`XX-X.md`) for each analyzed session.
- **Session Slide Decks**: PowerPoint files (`Session_XX.pptx`) containing A4-sized slides for each session and a group summary.
- **Index Dashboard**: [index2.html](./index2.html) - A consolidated dashboard for quick review and content copying for PPT slides.
- **Analysis Format**:
  1. Title
  2. Authors
  3. Affiliation
  4. Technical Implications
  5. Significance & Benchmarking against SDC technology levels

## Analyzed Sessions
- 13-1 ~ 13-4: AR/VR Display Systems
- 20-1 ~ 20-4: Micro-OLED and High PPI LCDs
- 44-1 ~ 44-4: Ultra-high PPI OLEDoS and Optics
- 56-1 ~ 56-4: Advanced Driving Circuits and LTPO
- 72-1 ~ 72-2: Micro-LED Integration and Driving
- 74-1 ~ 74-3: OLED Reliability and High-Resolution Patterning
- 91-1 ~ 91-4: Under-Display Camera (UDC) and IR Sensors
- 98-1 ~ 98-2: In-cell Touch and Self-Diagnosis

## How to Use
Open `index2.html` in any modern web browser to view the consolidated dashboard. For official presentations, use the `Session_XX.pptx` files which contain formatted A4-sized slides.
