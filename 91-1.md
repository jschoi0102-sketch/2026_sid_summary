# 91-1. Breaking the Under-Display Camera’s Dilemma Between Diffraction and Pixel Density Using Incoherent Pupil Synthesis

1. **제목**
   - Breaking the Under-Display Camera’s Dilemma Between Diffraction and Pixel Density Using Incoherent Pupil Synthesis

2. **저자**
   - Xinni Xie, Qimeng Wang, Yi Liu, Bo-Ru Yang, Zong Qin

3. **소속**
   - Sun Yat-Sen University (중국 중산대학교)

4. **시사점**
   - UDC(Under-Display Camera)의 해상도 유지와 화질 저하(회절 현상) 사이의 트레이드오프 문제를 해결하기 위해 '비간섭 동공 합성(Incoherent Pupil Synthesis)' 기술을 제안함.
   - 카메라 동공(Pupil) 영역을 나누어 서로 직교하는 편광판이나 시간차 셔터를 배치함으로써, 각 영역을 통과하는 빛이 서로 간섭하지 않게 함.
   - 각 영역에 대응하는 패널의 픽셀 구조를 서로 다르게 설계하여, 개별적으로는 화질 복원이 어려운 광학 전달 함수(OTF)의 빈틈(Gap)들을 합성을 통해 상호 보완적으로 제거함.
   - 결과적으로 고PPI 패널을 유지하면서도 회절에 의한 이미지 블러를 획기적으로 개선하여 선명한 이미지 복원이 가능함을 실험으로 증명함.

5. **본기술의 의의 당사 기술수준과 비교**
   - **의의**: UDC의 한계인 물리적 회절 현상을 패널의 투과율 개선이라는 정공법 대신, 광학적인 '비간섭 합성'이라는 신개념으로 우회하여 해결한 점이 매우 혁신적임. 특히 직교 편광판과 같이 구현이 용이한 부품을 활용하여 복원 가능한 정보량(OTF Magnitude)을 극대화한 것은 차세대 UDC 설계의 새로운 방향성을 제시함.
   - **기술수준 비교**: 당사는 세계 최초 UDC 상용화 브랜드로서 픽셀 미세화 및 AI 기반 화질 복원 기술에서 앞서 있으나, 중산대학교의 이번 연구는 패널 레이아웃과 카메라 광학계의 '공동 설계(Co-design)'를 통해 물리적 한계를 극복하려는 시도임. 당사의 차세대 UDC 전략에서도 단순히 투과율을 높이는 단계를 넘어, 렌즈 단의 특수한 필터(편광 등)와 연동된 비대칭 픽셀 설계 등을 통해 정보 손실을 최소화하는 '계산적 디스플레이(Computational Display)' 기술을 적극 벤치마킹하고 당사만의 고유 특허로 확보할 필요가 있음.
