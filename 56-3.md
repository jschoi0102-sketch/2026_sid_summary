# 56-3. Novel Double Rate Driving Pixel Structure for High-Performance 4K 165Hz OLED TVs with Reduced Number of Source Driver ICs

1. **제목**
   - Novel Double Rate Driving Pixel Structure for High-Performance 4K 165Hz OLED TVs with Reduced Number of Source Driver ICs

2. **저자**
   - Won-Ho Lee, Young-Min Jeong, Sang-Hoon Jung, Jun-Young Yang, Soo-Young Yoon

3. **소속**
   - Advanced Technology Laboratory, LG Display, Seoul, Korea

4. **시사점**
   - LG디스플레이(LGD)는 대면적 4K 165Hz OLED TV에서 드라이버 IC 개수를 절반으로 줄이면서도 고성능을 유지할 수 있는 혁신적인 DRD(Double Rate Driving) 픽셀 구조를 개발함.
   - 기존 DRD 방식은 인접한 서로 다른 색상의 화소를 하나의 데이터 라인에 연결하여 데이터 전환(Transition)이 빈번하고 이로 인한 발열(SD-IC 온도 상승)이 심각했으나, LGD는 '링크 브랜치(Link branch)'를 도입하여 동일 색상의 화소끼리 데이터 라인을 공유하도록 설계함.
   - 이를 통해 데이터 전환 횟수를 획기적으로 줄여 소스 드라이버 IC의 온도를 기존 DRD 대비 약 50% 낮추는 데 성공함.
   - 또한 '프레임 교차 구동(AFD, Alternate Frame Driving)' 기술(Z-구동과 S-구동을 프레임별로 교차)을 적용하여 특정 패턴(h1by1, dot pattern)에서 발생할 수 있는 휘도 불균일 문제를 해결함.

5. **본기술의 의의 당사 기술수준과 비교**
   - **의의**: 대형 OLED TV 시장에서 원가 경쟁력(IC 50% 감축)을 확보하면서도 고주사율(165Hz) 구현 시 발생하는 발열과 화질 저하 문제를 동시에 해결한 매우 실용적이고 강력한 솔루션임. 특히 드라이버 IC 발열을 절반으로 줄인 것은 제품의 신뢰성 향상과 슬림한 모듈 설계에 직접적인 기여를 함.
   - **기술수준 비교**: LGD는 대형 WOLED 패널의 구동 아키텍처 효율화 측면에서 독보적인 완성도를 보여주고 있음. 삼성디스플레이의 대면적 패널 전략에서도 프리미엄 라인업의 원가 경쟁력 제고를 위해 이러한 고도화된 DRD 기술 도입이 필수적임. LGD의 '동일 색상 화소 연결' 컨셉과 'AFD 알고리즘'은 당사의 대형 패널 설계 시 핵심적인 벤치마킹 포인트이며, 발열 제어와 화질 균일성 확보라는 두 마리 토끼를 잡기 위한 당사만의 차별화된 구동 IP 확보가 시급함.
